# TaskFlow

Sistema de gerenciamento de projetos inspirado em Trello, Jira e ClickUp.

## Tecnologias

### Backend
- Java 21
- Spring Boot
- Spring Security
- JWT
- PostgreSQL

### Frontend
- React
- Vite
- Material UI

## Objetivo

Criar uma plataforma para gerenciamento de projetos, tarefas e equipes utilizando metodologia ágil.
